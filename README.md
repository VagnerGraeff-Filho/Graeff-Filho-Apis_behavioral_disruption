# Graeff-Filho-Apis_behavioral_disruption
Repositore created to keep arquives related to manuscript: Graeff-Filho (2025)-  Behavioral Disruption in Honey Bees (Apis mellifera) Exposed to Isolated and Combined Insecticides
We have included the script containing all statistical analyses performed in R. The raw_spreadsheets folder contains the spreadsheet from the tracking analysis using the AnimalTA software, unmodified. It also contains a spreadsheet from the observation manual.
